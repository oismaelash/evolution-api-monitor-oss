# 🔁 HEALTH CHECK LOGIC

Every configurable interval (default: 5 minutes):

1. Send:

   * status check
   * presence check

2. If both succeed:
   → mark as HEALTHY (CONNECTED)

3. If failure occurs:
   → increment failure counter

4. If 3 consecutive failures:
   → trigger restart

5. Restart flow:

   * restart instance
   * wait 1 minute
   * run health check again

6. If still failing:
   → second restart attempt

7. After second failure:
   → send alert
